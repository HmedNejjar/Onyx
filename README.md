# Onyx ‚Äî GPT-Style Language Model

A from-scratch GPT implementation in PyTorch. Trains a causal transformer decoder on text data and generates continuations from a prompt using top-p sampling.

---

## What It Does

- Tokenizes raw text using the GPT-2 BPE tokenizer
- Trains a 12-layer transformer decoder (768-dim, 12 heads) on next-token prediction
- Generates text autoregressively from a prompt

## What Each File Does

**`model.py`** ‚Äî `Onyx(nn.Module)`  
Token + positional embeddings ‚Üí 12 transformer blocks ‚Üí LayerNorm ‚Üí linear projection to vocab logits. `generate()` tokenizes a prompt and samples tokens one at a time using `_sample_top_p()`.

**`TransformerBlock.py`** ‚Äî `TransformerBlock(nn.Module)`  
Pre-norm decoder block. Flow: `LayerNorm ‚Üí MultiheadAttention ‚Üí Dropout ‚Üí Residual`, then `LayerNorm ‚Üí FFN (Linear‚ÜíGELU‚ÜíLinear‚ÜíDropout) ‚Üí Residual`. Causal mask passed in from the model.

**`Dataloader.py`** ‚Äî `GPTDataset(Dataset)`  
Memory-maps a `.npy` token file. `__getitem__(i)` returns `(tokens[i*stride : i*stride+seq_len], tokens[i*stride+1 : ...+1])` ‚Äî input and target shifted by 1 for next-token prediction.

**`precompute.py`**  
Downloads WikiText-103 via HuggingFace `datasets`, tokenizes with GPT-2 tokenizer, saves `wikitext_train.npy` and `wikitext_val.npy`. Run once before training.

**`train.py`**  
Training loop with mixed precision (AMP via `GradScaler`), gradient accumulation (effective batch = `BATCH_SIZE √ó ACCUMULATION_STEPS`), gradient clipping (`max_norm=1.0`), and validation after each epoch. Saves checkpoint on accuracy improvement.

## How to Run

**1. Tokenize data** (once):
```bash
python GPT/precompute.py
```
Outputs `Data/wikitext_train.npy` and `Data/wikitext_val.npy`.

**2. Train:**
```bash
python GPT/train.py
```
Saves `best_model.pth` when validation accuracy improves.

**3. Generate:**
```python
from GPT.model import Onyx
import torch

model = Onyx()
model.load_state_dict(torch.load("best_model.pth"))
model.eval()

with torch.no_grad():
    print(model.generate("Once upon a time", max_tokens=100, top_p=0.9, temp=0.7))
```

## Key Hyperparameters

| Parameter | Default | Notes |
|---|---|---|
| `VOCAB_SIZE` | 50,257 | GPT-2 vocab |
| `CONTEXT_LEN` | 1024 | Max sequence length |
| `EMB_SIZE` | 768 | Embedding dimension |
| `NUM_HEADS` | 12 | Attention heads |
| `NUM_LAYERS` | 12 | Transformer blocks |
| `BATCH_SIZE` | 2 | Per-step GPU batch |
| `ACCUMULATION_STEPS` | 8 | Effective batch = 16 |
| `LR` | 1e-4 | AdamW learning rate |
| `STRIDE` | 256 | Sliding window step |
